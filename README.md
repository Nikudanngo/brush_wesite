# Reactで作ったwebサイト - 竹歯ブラシ - KAGUYA

使用したライブラリ [Create React App](https://github.com/facebook/create-react-app).

使用したapi [Material-UI](https://mui.com/)


# めも

- 現在実際に公開しているのはbuildのディレクトリ
- publicは使ってない

# 利用するコマンド

### `npm start`

開発者モードでアプリを実行する．\
[http://localhost:3000](http://localhost:3000)を開くとブラウザで表示される．\
変更を加えると自動的に再読み込みされる

### `npm run build`
    
本番用のアプリをビルドする.\
ついでにいろいろハッシュされて最適化される.
